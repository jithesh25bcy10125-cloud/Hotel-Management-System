# Problem Statement

Managing hotel rooms manually can become confusing and inefficient, especially when the number of rooms increases. Staff may struggle to track which rooms are available, which are booked, and what types of rooms exist. There is a need for a simple system that can handle basic hotel room operations without complexity.

This project solves that problem by providing a basic, menu-driven application that helps hotel staff manage rooms easily.




 # Scope of the Project

The scope of this project is limited to the essential operations required for small hotels, lodges, or guest houses. It includes:

Adding new rooms with details

Searching for a room

Booking and checking out rooms

Viewing all rooms with their status


The project focuses only on room management and does not include advanced modules such as payment processing, customer records, or online integration.



 # Target Users

This system is mainly designed for:

Hotel staff

Lodge and guest-house receptionists

Small accommodation businesses

Students learning Python-based project development


It aims to provide a simple and beginner-friendly tool to manage room details without any technical complexity.




 # High-Level Features

Add Room: Store room number, type, and price

Search Room: Quickly find rooms using the room number

Book Room: Mark rooms as booked only if they are available

Checkout Room: Free a booked room

Display All Rooms: View all room details with their status

Menu-Driven Interface: Easy to navigate for any user
